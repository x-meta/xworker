<p style="widows: 2; text-transform: none; text-indent: 0px; font: medium Simsun; white-space: normal; orphans: 2; letter-spacing: normal; color: rgb(0,0,0); word-spacing: 0px; -webkit-text-size-adjust: auto; -webkit-text-stroke-width: 0px">FilterSets are groups of filters. Filters can be defined as token-value pairs or be read in from a file. FilterSets can appear inside tasks that support this feature or at the same level as<span class="Apple-converted-space">&nbsp;</span><code>&lt;target&gt;</code><span class="Apple-converted-space">&nbsp;</span>- i.e., as children of<span class="Apple-converted-space">&nbsp;</span><code>&lt;project&gt;</code>.</p>
<p style="widows: 2; text-transform: none; text-indent: 0px; font: medium Simsun; white-space: normal; orphans: 2; letter-spacing: normal; color: rgb(0,0,0); word-spacing: 0px; -webkit-text-size-adjust: auto; -webkit-text-stroke-width: 0px">FilterSets support the<span class="Apple-converted-space">&nbsp;</span><code>id</code><span class="Apple-converted-space">&nbsp;</span>and<span class="Apple-converted-space">&nbsp;</span><code>refid</code><span class="Apple-converted-space">&nbsp;</span>attributes. You can define a FilterSet with an<span class="Apple-converted-space">&nbsp;</span><code>id</code><span class="Apple-converted-space">&nbsp;</span>attribute and then refer to that definition from another FilterSet with a<span class="Apple-converted-space">&nbsp;</span><code>refid</code><span class="Apple-converted-space">&nbsp;</span>attribute. It is also possible to nest filtersets into filtersets to get a set union of the contained filters.</p>
<p style="widows: 2; text-transform: none; text-indent: 0px; font: medium Simsun; white-space: normal; orphans: 2; letter-spacing: normal; color: rgb(0,0,0); word-spacing: 0px; -webkit-text-size-adjust: auto; -webkit-text-stroke-width: 0px">In addition, FilterSets can specify<span class="Apple-converted-space">&nbsp;</span><code>begintoken</code><span class="Apple-converted-space">&nbsp;</span>and/or<span class="Apple-converted-space">&nbsp;</span><code>endtoken</code><span class="Apple-converted-space">&nbsp;</span>attributes to define what to match.</p>
<p style="widows: 2; text-transform: none; text-indent: 0px; font: medium Simsun; white-space: normal; orphans: 2; letter-spacing: normal; color: rgb(0,0,0); word-spacing: 0px; -webkit-text-size-adjust: auto; -webkit-text-stroke-width: 0px">Filtersets are used for doing replacements in tasks such as<span class="Apple-converted-space">&nbsp;</span><code>&lt;copy&gt;</code>, etc.</p>
<p style="widows: 2; text-transform: none; text-indent: 0px; font: medium Simsun; white-space: normal; orphans: 2; letter-spacing: normal; color: rgb(0,0,0); word-spacing: 0px; -webkit-text-size-adjust: auto; -webkit-text-stroke-width: 0px">If you specify multiple values for the same token, the last one defined within a filterset will be used.</p>
<p style="widows: 2; text-transform: none; text-indent: 0px; font: medium Simsun; white-space: normal; orphans: 2; letter-spacing: normal; color: rgb(0,0,0); word-spacing: 0px; -webkit-text-size-adjust: auto; -webkit-text-stroke-width: 0px"><strong>Note:<span class="Apple-converted-space">&nbsp;</span></strong>When a filterset is used in an operation, the files are processed in text mode and the filters applied line by line. This means that the copy operations will typically corrupt binary files. When applying filters you should ensure that the set of files being filtered are all text files.</p>
