<p>基本命题知识库，用于存储基本的命题知识。</p>

<p><strong>语法：</strong></p>

<pre style="text-align: left;">
 语句            -&gt; 原子语句 : 复合语句
 原子语句         -&gt; True : False : P : Q : R : ... // (1)
 复合语句         -&gt; (语句) | [语句]
                 :  ~语句                //否定
                 :  语句 &amp; 语句           //并且
                 :  语句 | 语句           //或者
                 :  语句 =&gt; 语句          //蕴含，推出
                 :  语句 &lt;=&gt; 语句         //等价，而这可以相互推出
 
 操作符优先级: ~, &amp;, |, =&gt;, &lt;=&gt; // (2)
 </pre>

<p><span style="background-color:rgb(255, 255, 255); color:rgb(71, 71, 71); float:none; text-align:left">注释&nbsp;(1): 书本上要求符号用大写字母，但在我们的实现里可以用任何字符，包括单词。</span><br style="text-align: left;" />
<br style="text-align: left;" />
<span style="background-color:rgb(255, 255, 255); color:rgb(71, 71, 71); float:none; text-align:left">注释&nbsp;(2): 实现是右结合的。</span></p>

<pre style="text-align: left;">
 A &amp; B &amp; C &amp; D 
 
 will be parsed as:
 
 (A &amp; (B &amp; (C &amp; D)))</pre>

<p><strong>可以重写的方法：</strong></p>

<ul>
	<li><strong>String getSentences()</strong><br />
	返回语句列表。</li>
</ul>
