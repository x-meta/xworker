<h3>事物&nbsp;Example</h3>

<p>&nbsp;&nbsp;&nbsp; 一个事物是通过其描述者(descriptor）创建出来的，因此在事物的演示里可以看到事物本身以及它的描述者。</p>

<h3>操作说明</h3>

<ul>
	<li>左面的菜单节点选择要查看的示例。</li>
	<li>在右面标签页的第一项是事物的编辑界面。</li>
	<li>在右面标签页的第二项是事物的描述者的编辑界面。</li>
	<li>在右面标签页的第三项是事物和描述和继承关系。</li>
</ul>

<h3><span style="color:rgb(255, 0, 0)">重要提示</span></h3>

<p><span style="color:rgb(255, 0, 0)">&nbsp;&nbsp;&nbsp; 在演示中可以修改事物，而XWorker中的任何事物都可以被修改，包括XWorker本身，并且事物修改后会即时生效，因此如果修改了基础事物那么可能会导致系统崩溃或出现异常。</span></p>

<p><span style="color:rgb(255, 0, 0)">&nbsp;&nbsp;&nbsp; 如果出现异常可以重新安装XWorker或从备份中恢复被修改的事物。</span></p>

<h3>Thing&nbsp;Examples</h3>

<p>&nbsp;&nbsp;&nbsp; May be first lean the basic concept of thing, include how to create and edit thing, see XWorker tutorial.</p>

<h3>Operate</h3>

<ul>
	<li>Click tree menu node select example.</li>
	<li>Right first TabItem is the example thing editor.</li>
	<li>Right second TabItem is the descriptor(class) thing of the example thing.</li>
	<li>Right third TabItem is all descriptor&nbsp;and extended things of the example thing.</li>
</ul>

<h3><span style="color:rgb(255, 0, 0)">Important</span></h3>

<p><span style="color:rgb(255, 0, 0)">&nbsp;&nbsp;&nbsp; Can modify things in example, and any thing can be modify in xworker, include xworker itself， so be carefull, if the base thing modified xwoker may be crash.</span></p>

<p><span style="color:rgb(255, 0, 0)">&nbsp;&nbsp;&nbsp; If crashed you can reinstall xworker or copy the orignal thing and replace the modified thing.</span></p>
