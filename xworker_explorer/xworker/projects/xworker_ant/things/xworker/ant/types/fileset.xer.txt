<p style="widows: 2; text-transform: none; text-indent: 0px; font: medium Simsun; white-space: normal; orphans: 2; letter-spacing: normal; color: rgb(0,0,0); word-spacing: 0px; -webkit-text-size-adjust: auto; -webkit-text-stroke-width: 0px">A FileSet is a group of files. These files can be found in a directory tree starting in a base directory and are matched by patterns taken from a number of<span class="Apple-converted-space">&nbsp;</span><a href="http://ant.apache.org/manual/Types/patternset.html">PatternSets</a><span class="Apple-converted-space">&nbsp;</span>and<span class="Apple-converted-space">&nbsp;</span><a href="http://ant.apache.org/manual/Types/selectors.html">Selectors</a>.</p>
<p style="widows: 2; text-transform: none; text-indent: 0px; font: medium Simsun; white-space: normal; orphans: 2; letter-spacing: normal; color: rgb(0,0,0); word-spacing: 0px; -webkit-text-size-adjust: auto; -webkit-text-stroke-width: 0px">PatternSets can be specified as nested<span class="Apple-converted-space">&nbsp;</span><code>&lt;patternset&gt;</code><span class="Apple-converted-space">&nbsp;</span>elements. In addition, FileSet holds an implicit PatternSet and supports the nested<span class="Apple-converted-space">&nbsp;</span><code>&lt;include&gt;</code>,<span class="Apple-converted-space">&nbsp;</span><code>&lt;includesfile&gt;</code>,<span class="Apple-converted-space">&nbsp;</span><code>&lt;exclude&gt;</code><span class="Apple-converted-space">&nbsp;</span>and<span class="Apple-converted-space">&nbsp;</span><code>&lt;excludesfile&gt;</code><span class="Apple-converted-space">&nbsp;</span>elements of PatternSet directly, as well as PatternSet's attributes.</p>
<p style="widows: 2; text-transform: none; text-indent: 0px; font: medium Simsun; white-space: normal; orphans: 2; letter-spacing: normal; color: rgb(0,0,0); word-spacing: 0px; -webkit-text-size-adjust: auto; -webkit-text-stroke-width: 0px">Selectors are available as nested elements within the FileSet. If any of the selectors within the FileSet do not select the file, the file is not considered part of the FileSet. This makes a FileSet equivalent to an<span class="Apple-converted-space">&nbsp;</span><code>&lt;and&gt;</code><span class="Apple-converted-space">&nbsp;</span>selector container.</p>
