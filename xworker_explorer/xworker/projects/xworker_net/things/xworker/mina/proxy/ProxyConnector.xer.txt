<p>ProxyConnector.java - Decorator for <a title="interface in org.apache.mina.transport.socket" href="http://mina.apache.org/report/trunk/apidocs/org/apache/mina/transport/socket/SocketConnector.html"><code>SocketConnector</code></a> to provide proxy support, as suggested by MINA list discussions.</p>
<p>Operates by intercepting connect requests and replacing the endpoint address with the proxy address, then adding a <a title="class in org.apache.mina.proxy.filter" href="http://mina.apache.org/report/trunk/apidocs/org/apache/mina/proxy/filter/ProxyFilter.html"><code>ProxyFilter</code></a> as the first <a title="interface in org.apache.mina.core.filterchain" href="http://mina.apache.org/report/trunk/apidocs/org/apache/mina/core/filterchain/IoFilter.html"><code>IoFilter</code></a> which performs any necessary handshaking with the proxy before allowing data to flow normally. During the handshake, any outgoing write requests are buffered.</p>
