XWorker
============
## 项目简介
XWorker是动态模型的开发平台。动态模型是一种编程方法，它使用模型来编程，模型可以用XML来表示。使用XWorker可以编辑、运行和发布模型程序。
## 动态模型简介
动态模型是从框架编程演化
详情可以参看https://www.xworker.org。
