<p><strong>作用</strong></p>

<p>用于生成JavaScript代码片段，所有能够生成JavaScript代码片段的都应注册到此子事物下。</p>

<ul>
	<li><strong>toHtml</strong><br />
	如果放在一般的HTML事物节点下，那么生成一个&lt;script type=&quot;javascript&quot;&gt;的标签，然后把代码片段放到这个标签下。<br />
	&nbsp;</li>
	<li><strong>toJavaScriptCode</strong><br />
	如果放在其他JavaScript的节点下，那么只会调用toJavaScriptCode方法生成JavaScript代码片段。</li>
</ul>

<p><strong>JavaScript注册</strong></p>

<p>所有能够生成 JavaScript代码的事物都应该注册到本事物下。</p>

<p><strong>重写toHtml方法</strong></p>

<p>如果一个事物继承了JavaScript并且只是用于生成JavaScript代码的，那么应该重写toHtml方法，返回在toHtml方法里返回null，可以使用NOOP动作。</p>
