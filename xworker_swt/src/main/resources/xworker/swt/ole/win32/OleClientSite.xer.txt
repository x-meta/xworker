#$@text#$@
<p>OleClientSite provides a site to manage an embedded OLE Document within a container.</p>
<p>The OleClientSite provides the following capabilities:</p>
<ul>
    <li>creates the in-place editor for a blank document or opening an existing OLE Document</li>
    <li>lays the editor out</li>
    <li>provides a mechanism for activating and deactivating the Document</li>
    <li>provides a mechanism for saving changes made to the document</li>
</ul>
<p>This object implements the OLE Interfaces IUnknown, IOleClientSite, IAdviseSink, IOleInPlaceSite</p>
<p>Note that although this class is a subclass of <code>Composite</code>, it does not make sense to add <code>Control</code> children to it, or set a layout on it.</p>
