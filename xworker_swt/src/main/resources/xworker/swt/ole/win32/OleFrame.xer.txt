#$@text#$@
<p>OleFrame is an OLE Container's top level frame.</p>
<p>This object implements the OLE Interfaces IUnknown and IOleInPlaceFrame</p>
<p>OleFrame allows the container to do the following:</p>
<ul>
    <li>position and size the ActiveX Control or OLE Document within the application</li>
    <li>insert menu items from the application into the OLE Document's menu</li>
    <li>activate and deactivate the OLE Document's menus</li>
    <li>position the OLE Document's menu in the application</li>
    <li>translate accelerator keystrokes intended for the container's frame</li>
</ul>
