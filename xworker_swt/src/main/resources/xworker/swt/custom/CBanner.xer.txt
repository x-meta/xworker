Sdescription
#$@text#$@
<p>Instances of this class implement a Composite that lays out its children and allows programmatic control of the layout. It draws a separator between the left and right children which can be dragged to resize the right control. CBanner is used in the workbench to layout the toolbar area and perspective switching toolbar.</p>
<p>Note that although this class is a subclass of <code>Composite</code>, it does not make sense to set a layout on it.</p>
