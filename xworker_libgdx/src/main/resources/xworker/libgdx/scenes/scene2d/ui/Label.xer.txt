<p>A text label, with optional word wrapping.</p>

<p>Unlike most scene2d.ui widgets, label can be scaled and rotated using the actor&#39;s scale, rotation, and origin. Note that a scaled or rotated label causes a SpriteBatch flush when it is drawn, so should be used relatively sparingly.</p>
