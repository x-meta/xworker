<p style="text-align: left;">Unlike most scene2d.ui widgets, label can be scaled and rotated using the actor&#39;s scale, rotation, and origin. This only affects drawing, other scene2d.ui widgets will still use the unscaled and unrotated bounds of the label. Note that a scaled or rotated label causes a SpriteBatch flush when it is drawn, so should be used relatively sparingly.</p>
