#$@text#$@
<p><span style="background-color:rgb(255, 255, 255); color:rgb(53, 56, 51)">A skin stores resources for UI widgets to use (texture regions, ninepatches, fonts, colors, etc). Resources are named and can be looked up by name and type. Resources can be described in JSON. Skin provides useful conversions, such as allowing access to regions in the atlas as ninepatches, sprites, drawables, etc. The get* methods return an instance of the object in the skin. The new* methods return a copy of an instance in the skin.</span></p>

<p style="text-align: left;">See the <a href="https://code.google.com/p/libgdx/wiki/Skin" style="color: blue; text-decoration: none;">documentation</a> for more.</p>
