FileSets are groups of files. These files can be found in a directory tree starting in a base directory and are matched by patterns taken from a number of <a href="file.///C./zyx/ANT/docs/manual/CoreTypes/patternset.html">PatternSets</a> and <a href="file///C/zyx/ANT/docs/manual/CoreTypes/selectors.html">Selectors</a>. FileSets can appear inside tasks that support this feature or at the same level as <code><font face="新宋体">target</font></code> - i.e., as children of <code><font face="新宋体">project</font></code>.<br /><br />PatternSets can be specified as nested <code><font face="新宋体">&lt;patternset&gt;</font></code> elements. In addition, FileSet holds an implicit PatternSet and supports the nested <code><font face="新宋体">&lt;include&gt;</font></code>, <code><font face="新宋体">&lt;includesfile&gt;</font></code>, <code><font face="新宋体">&lt;exclude&gt;</font></code> and <code><font face="新宋体">&lt;excludesfile&gt;</font></code> elements of PatternSet directly, as well as PatternSet's attributes.<br /><br />Selectors are available as nested elements.within the FileSet. If any of the selectors within the FileSet do not select the file, the file is not considered part of the FileSet. This makes FileSets equivalent to an <code><font face="新宋体">&lt;and&gt;</font></code> selector container.
