#$@text#$@
<p>An implementation of <a title="interface in org.eclipse.draw2d" href="http://help.eclipse.org/galileo/topic/org.eclipse.draw2d.doc.isv/reference/api/org/eclipse/draw2d/Connection.html"><code>Connection</code></a> based on Polyline. PolylineConnection adds the following additional features:</p>
<ul>
    <li>A <a title="interface in org.eclipse.draw2d" href="http://help.eclipse.org/galileo/topic/org.eclipse.draw2d.doc.isv/reference/api/org/eclipse/draw2d/ConnectionRouter.html"><code>ConnectionRouter</code></a> may be provided which will be used to determine the connections points.</li>
    <li>Children may be added. The bounds calculation is extended such that the bounds is the smallest Rectangle which is large enough to display the Polyline and all of its children figures.</li>
    <li>A <a title="class in org.eclipse.draw2d" href="http://help.eclipse.org/galileo/topic/org.eclipse.draw2d.doc.isv/reference/api/org/eclipse/draw2d/DelegatingLayout.html"><code>DelegatingLayout</code></a> is set as the default layout. A delegating layout allows children to position themselves via <a title="interface in org.eclipse.draw2d" href="http://help.eclipse.org/galileo/topic/org.eclipse.draw2d.doc.isv/reference/api/org/eclipse/draw2d/Locator.html"><code>Locators</code></a>.</li>
</ul>
