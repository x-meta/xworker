<p>The input parameters of the script. <br />
<br />
In the processing process, data is passed through a context similar to Map. The input parameters can specify which parameters need to be passed in. The input parameters are optional. After specifying the input parameters, the parameters can be verified. If you choose to verify, then in the case of verification errors The script program will not be executed. <br />
<br />
Context and input parameters are used for data input before executing the program, but the context is actively acquiring input data, and input parameters are passively input. For example, the session of the database is obtained through the context, and the parameters submitted by the web form are passed in passively. </p>
