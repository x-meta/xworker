<p><strong>XWorker的特点</strong></p>

<ul>
	<li><strong>开放</strong><br />
	XWorker基于<a href="http://www.xworker.org/do?sc=xworker.app.orgweb.web.Index&amp;id=7" style="line-height: 20.79px;">开源协议Apache2.0</a>，代码和功能都是开放的，其次XWorker就是使用XWorker开发的，每个人都可以改变它，XWorker将是一个汇集大家的智慧的结晶。<br />
	&nbsp;</li>
	<li><strong>易创</strong><br />
	XWorker是一个编程平台，它使用了基于数据的配置编程方法，通过简单的配置就可以创建出复杂的应用。<br />
	<a href="http://www.xworker.org/do?sc=xworker.app.orgweb.web.Index&amp;id=452">进入编程</a>。<br />
	&nbsp;</li>
	<li><strong>易享</strong><br />
	XWorker也是一个分享编程工具和经验的平台，可以通过简单的方式分享类库、笔记、示例、工具、模板和经验等。<br />
	<a href="http://www.xworker.org/do?sc=xworker.app.orgweb.web.Index&amp;id=540">进入分享</a>。<br />
	<strong>&nbsp;</strong></li>
	<li><strong>易用</strong><br />
	XWorker可以把所有人分享的类库、经验、工具和应用等无缝的完美的结合起来，使其成为一个用的人越多功能也就越多的应用平台。</li>
</ul>
