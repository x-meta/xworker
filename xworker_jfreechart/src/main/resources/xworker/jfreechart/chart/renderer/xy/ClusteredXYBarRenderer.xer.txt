#$@text#$@
<div class="block">An extension of <a title="class in org.jfree.chart.renderer.xy" href="http://www.jfree.org/jfreechart/api/javadoc/org/jfree/chart/renderer/xy/XYBarRenderer.html"><code>XYBarRenderer</code></a> that displays bars for different series values at the same x next to each other. The assumption here is that for each x (time or else) there is a y value for each series. If this is not the case, there will be spaces between bars for a given x. The xworker.example shown here is generated by the <code>ClusteredXYBarRendererDemo1.java</code> program included in the JFreeChart demo collection: <br />
<br />
<img alt="ClusteredXYBarRendererSample.png" src="http://www.jfree.org/jfreechart/api/javadoc/images/ClusteredXYBarRendererSample.png" />
<p>This renderer does not include code to calculate the crosshair point for the plot.</p>
</div>
