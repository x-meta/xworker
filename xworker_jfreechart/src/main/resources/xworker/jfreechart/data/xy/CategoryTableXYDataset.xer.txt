<div class="block">
<div class="block">An implementation variant of the <a title="interface in org.jfree.data.xy" href="http://www.jfree.org/jfreechart/api/javadoc/org/jfree/data/xy/TableXYDataset.html"><code>TableXYDataset</code></a> where every series shares the same x-values (required for generating stacked area charts). This implementation uses a <a title="class in org.jfree.data" href="http://www.jfree.org/jfreechart/api/javadoc/org/jfree/data/DefaultKeyedValues2D.html"><code>DefaultKeyedValues2D</code></a> Object as backend implementation and is hence more &quot;category oriented&quot; than the <a title="class in org.jfree.data.xy" href="http://www.jfree.org/jfreechart/api/javadoc/org/jfree/data/xy/DefaultTableXYDataset.html"><code>DefaultTableXYDataset</code></a> implementation.
<p>This implementation provides no means to remove data items yet. This is due to the lack of such facility in the DefaultKeyedValues2D class.</p>
<p>This class also implements the <a title="interface in org.jfree.data.xy" href="http://www.jfree.org/jfreechart/api/javadoc/org/jfree/data/xy/IntervalXYDataset.html"><code>IntervalXYDataset</code></a> interface, but this implementation is provisional.</p>
