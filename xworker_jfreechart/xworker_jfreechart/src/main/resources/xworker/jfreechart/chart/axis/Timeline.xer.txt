#$@text#$@
<div class="block">An interface that defines the contract for a Timeline.
<p>A Timeline will present a series of values to be used for an axis. Each Timeline must provide transformation methods between domain values and timeline values. In theory many transformations are possible. This interface has been implemented completely in <a title="class in org.jfree.chart.axis" href="http://www.jfree.org/jfreechart/api/javadoc/org/jfree/chart/axis/SegmentedTimeline.html"><code>SegmentedTimeline</code></a>.</p>
<p>A timeline can be used as parameter to a <a title="class in org.jfree.chart.axis" href="http://www.jfree.org/jfreechart/api/javadoc/org/jfree/chart/axis/DateAxis.html"><code>DateAxis</code></a> to define the values that this axis supports. As an xworker.example, the <a title="class in org.jfree.chart.axis" href="http://www.jfree.org/jfreechart/api/javadoc/org/jfree/chart/axis/SegmentedTimeline.html"><code>SegmentedTimeline</code></a> implements a timeline formed by segments of equal length (ex. days, hours, minutes) where some segments can be included in the timeline and others excluded. Therefore timelines like &quot;working days&quot; or &quot;working hours&quot; can be created where non-working days or non-working hours respectively can be removed from the timeline, and therefore from the axis. This creates a smooth plot with equal separation between all included segments.</p>
<p>Because Timelines were created mainly for Date related axis, values are represented as longs instead of doubles. In this case, the domain value is just the number of milliseconds since January 1, 1970, 00.00.00 GMT as defined by the getTime() method of <code>Date</code>.</p>
</div>

import org.jfree.chart.axis.SegmentedTimeline;

import ognl.Ognl;

if(self.type == "Var"){
    return Ognl.getValue(self.varName, actionContext);
}

def segmentSize = self.getLong("segmentSize");
switch(self.segmentSize){
    case "DAY_SEGMENT_SIZE":
        segmentSize = SegmentedTimeline.DAY_SEGMENT_SIZE;
    case "FIFTEEN_MINUTE_SEGMENT_SIZE":
        segmentSize = SegmentedTimeline.FIFTEEN_MINUTE_SEGMENT_SIZE;
    case "HOUR_SEGMENT_SIZE":
        segmentSize = SegmentedTimeline.HOUR_SEGMENT_SIZE;
    case "MINUTE_SEGMENT_SIZE":
        segmentSize = SegmentedTimeline.MINUTE_SEGMENT_SIZE;
}

return new SegmentedTimeline(segmentSize, self.getInt("segmentsIncluded"), self.getInt("segmentsExcluded"));
