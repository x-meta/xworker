<div class="block">Renderer for drawing min max plot. This renderer draws all the series under the same category in the same x position using <code>objectIcon</code> and a line from the maximum value to the minimum value. For use with the <a title="class in org.jfree.chart.plot" href="http://www.jfree.org/jfreechart/api/javadoc/org/jfree/chart/plot/CategoryPlot.html"><code>CategoryPlot</code></a> class.
