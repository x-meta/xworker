<p>函数请求，当函数不完整时请求交互界面。</p>

<p><strong>函数的找寻顺序：</strong></p>

<ol>
	<li>通过functionVarName从actionContext中找。</li>
	<li>通过functionDescriptor创建一个函数事物。</li>
	<li>如果findFunction=true，那么提交一个新的找寻函数的界面。</li>
</ol>

<p><strong>函数的原理</strong></p>

<ul>
	<li><strong>本质</strong><br />
	函数UI的本质是确定参数，当参数确定了函数就可以执行了。<br />
	&nbsp;</li>
	<li><strong>回调</strong><br />
	参数也是通过函数生成的，因此当确定一个参数时会调用新的函数，新的函数执行完后会回调原函数传给的回调函数。<br />
	回调函数的类是FunctionCallback，执行其中的run方法，参数为函数的执行结果。</li>
</ul>
