#$@text#$@
<p>函数对话中的上下文，上下文相当于语境、会话主题等。</p>

<p><strong>主要功能：</strong></p>

<ol>
	<li><strong>设置和读取变量</strong><br />
	从函数对话上下文中设置和读取变量。<br />
	&nbsp;</li>
	<li><strong>充当话题</strong><br />
	设置了一个函数对话上下文后，在界面上会显示当前的话题。<br />
	&nbsp;</li>
	<li><strong>功能的索引</strong><br />
	选定了一个函数对话上下文后，可以查看相关的功能索引。</li>
</ol>

<p><strong>可以重写的方法：</strong></p>

<ul>
	<li><strong>getVars</strong><br />
	获取变量列表List&lt;Thing&gt;。<br />
	&nbsp;</li>
	<li><strong>getVarInputFunction</strong><br />
	获取指定变量名的输入函数，参数：name，返回字符串。<br />
	&nbsp;</li>
	<li><strong>getNeedInputVars</strong><br />
	获取需要输入的变量，返回List&lt;Thing&gt;，输入：vars（List&lt;String&gt;），可以为null。<br />
	&nbsp;</li>
	<li><strong>setVar</strong><br />
	设置变量，参数:name, value（Object）。<br />
	&nbsp;</li>
	<li><strong>getVar</strong><br />
	获取变量，参数：name。<br />
	&nbsp;</li>
	<li><strong>getFunctions</strong><br />
	获取功能列表，返回List&lt;Thing&gt;。<br />
	&nbsp;</li>
	<li><strong>addVar</strong><br />
	添加一个变量，参数：name(String)，返回Thing。<br />
	&nbsp;</li>
	<li><strong>addFunction</strong><br />
	添加一个功能，参数：name(String)，返回Thing。<br />
	&nbsp;</li>
	<li><strong>removeVar</strong><br />
	移除一个变量，参数：name(String)。<br />
	&nbsp;</li>
	<li><strong>removeFunction</strong><br />
	移除一个功能，参数：name(String)。</li>
</ul>
