<p>打开创建新事物的对话框。</p>

<p><strong>返回值：</strong>void。</p>

<p><strong>事件：</strong></p>

<ul>
	<li><strong>ok(Thing thing）</strong><br />
	如果新事物创建成功了。<br />
	&nbsp;</li>
	<li><strong>cancel()</strong><br />
	如果新事物创建失败或取消了。</li>
</ul>

<p><strong>可以重写的方法：</strong></p>

<ul>
	<li><strong>Map&lt;String, Object&gt; getAttributes()</strong><br />
	返回新事物额外自动设置的属性。默认从attributes属性获取，格式类似Properties的格式。<br />
	&nbsp;</li>
	<li><strong>String getThingName()</strong><br />
	返回事物名。默认从thingName属性获取，<a href="do?sc=xworker.ide.worldexplorer.swt.http.ThingDoc/@desc&amp;thing=xworker.lang.actions.ActionUtil/@GetString">参看实现文档</a>。<br />
	&nbsp;</li>
	<li><strong>String getDescriptor()</strong><br />
	返回描述者的路径。默认从descriptor属性获取，<a href="http://localhost:9001/do?sc=xworker.ide.worldexplorer.swt.http.ThingDoc/@desc&amp;thing=xworker.lang.actions.ActionUtil/@GetString">参看实现文档</a>。<br />
	&nbsp;</li>
	<li><strong>Shell getShell()</strong><br />
	返回用于创建对话框的父Shell，默认取IDE的shell。<a href="do?sc=xworker.lang.actions.doc.QuickSelfActionImpl/@SimpleControl#GetObject">参看文档</a>。</li>
</ul>

    def thing = new Thing("xworker.swt.xwidgets.JavaSourceViewer");
    thing.set("className", className);
    thing.set("icon", "icons/file/java.gif");
    def index = className.lastIndexOf(".");
    def javaName = null;
    if(index != -1 ){
        javaName = className.substring(index + 1, className.length()); 
    }else{
        javaName = className;
    }
    javaName = javaName + ".java";
    thing.set("name", javaName);
    def params = [:];
    params.put("path", "JavaSourceViewere: " + className);
    params.put("title", javaName);
    
    XWorkerUtils.ideOpenComposite(thing, params);
}else{
    log.info("Can not view java source, className is null, path=" + self.getMetadata().getPath());
}
