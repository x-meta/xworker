<p>Http下载器，下载指定URL的内容并保存为文件。</p>

<p><strong>UR列表</strong></p>

<p>&nbsp;&nbsp;&nbsp; 可以通过urls指定，也可以通过addUrls方法增加新的URL。</p>

<p><strong>可以重写的方法</strong></p>

<ul>
	<li><strong>List&lt;String&gt; getUrls(Thing self)</strong><br />
	返回要下载的URL列表，返回类型List&lt;String&gt;。<br />
	&nbsp;</li>
	<li><strong>String getFileName(HttpEntity entity, URI uri, String url, HttpResponse response, Thing self)</strong><br />
	返回要保存的文件名。<br />
	&nbsp;</li>
	<li><strong>onFinished(HttpEntity entity, URI uri, String url, HttpResponse response, Thing self, File file) </strong><br />
	当一个文件下载结束时的事件处理。<br />
	&nbsp;</li>
	<li><strong>onFailure(Thing self, Exception exception);</strong><br />
	当一个文件下载失败是的事件处理。</li>
</ul>

<p><strong>可以调用的方法</strong></p>

<ul>
	<li>addUrls(List&lt;String&gt; urls)<br />
	添加要下载的地址列表。</li>
</ul>

<p><strong>动作集合</strong></p>

<p>&nbsp;&nbsp;&nbsp; 通过动作可以定义事物的行为，事物的行为即动作，可以用多种语言和方式编写。</p>

<p><strong>self变量</strong></p>

<p>&nbsp;&nbsp;&nbsp; 当一个动作作为一个事物的行为执行时，会把该事物以self变量传给动作。</p>

<p><strong>第一个动作集合生效</strong></p>

<p>&nbsp;&nbsp;&nbsp; 只有事物的第一个actions子节点才被认为是行为的定义集合，第二个及以后不算。</p>
