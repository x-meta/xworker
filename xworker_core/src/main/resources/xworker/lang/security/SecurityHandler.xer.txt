<p>抽象的安全处理器，默认实现了注册等功能。其它安全处理器可以继承本模型，继承后需要根据情况实现create方法。</p>

<p><strong>需要实现的方法：</strong></p>

<ul>
	<li><strong>boolean doCheck(String env, String permission, String action, String path)</strong><br />
	处理权限请求，返回允许或不允许。默认返回false。</li>
</ul>

<p><strong>可以重写的方法：</strong></p>

<ul>
	<li><strong>String getPermission()</strong><br />
	返回可以处理的权限名。如果为null，将注册到环境上。<br />
	&nbsp;</li>
	<li><strong>String getAction()</strong><br />
	返回可以处理的动作名。如果action和path为null，注册到对应的Permission上。<br />
	&nbsp;</li>
	<li><strong>String getPath()</strong><br />
	返回可以处理的路径名，path为正则表达式。<br />
	&nbsp;</li>
	<li><strong>String getEnv()</strong><br />
	返回校验的环境，null或空表示本安全处理器无效。</li>
</ul>
