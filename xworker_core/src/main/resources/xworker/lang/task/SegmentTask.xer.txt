#$@text#$@
<p>分段任务。把一个大任务分为小段来并行处理。</p>

<p><strong>基本概念</strong></p>

<ul>
	<li><strong>段（Range)</strong><br />
	分段任务是对数字范围来分段的。比0~10按照步骤（Step)1来分段，可以分成10段。每次一段都有一个开始（start)和结束（end)，分段任务可以针对段做相应的操作。<br />
	注意：段应该是start &lt;= range &lt; end，即大于等于开始，而小于结束。<br />
	&nbsp;</li>
	<li><strong>段生成器(RangeGenerator)</strong><br />
	用于生成段和把数字转成对象。比如时间段生成器可以按照天、月等生成段，并把开始和结束转化为时间对象。<br />
	&nbsp;</li>
	<li><strong>段存储器(RangeStore)</strong><br />
	段存储器用于保存段的数据。使用持久化的段段存储器可以在重新启动分段任务时恢复先前的状态。</li>
</ul>

<p><strong>使用方法</strong></p>

<p>&nbsp;&nbsp;&nbsp; 设置段生成器和段存储器，重写<strong>boolean doRange(SegmentTask task，Range range, Object start, Object end)</strong>方法。</p>

<p>&nbsp;&nbsp;&nbsp; 一个分段任务同一时刻只有一个实例，启动任务执行run方法，停止用stop方法。</p>

<p><strong>可以重写的方法</strong></p>

<ul>
	<li><strong>boolean doRange(SegmentTask task，Range range, Object start, Object end)</strong><br />
	执行段的操作。返回true表示处理成功，返回false表示处理失败。</li>
</ul>

<p><strong>可以监听的事件</strong></p>

<ul>
	<li><strong>void onRangeSuccess(SegmentTask task，Range range, Callable&lt;Boolean&gt; callable)</strong></li>
	<li><strong>void onRangeException(SegmentTask task，Range range, Callable&lt;Boolean&gt; callable, Exception e)</strong></li>
	<li><strong>void onRangeFailed(SegmentTask task，Range range, Callable&lt;Boolean&gt; callable)</strong></li>
	<li><strong>void onException(SegmentTask task，Exception e)</strong></li>
	<li><strong>void onFinished(SegmentTask task)</strong></li>
	<li><strong>void onRangeTimeout(SegmentTask task，Range range, Callable&lt;Boolean&gt; callable)</strong></li>
</ul>
