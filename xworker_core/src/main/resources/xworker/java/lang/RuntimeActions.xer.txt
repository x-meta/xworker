<p>执行一个系统的进程，如果使用同步sync，那么在进程结束后把进程的输入作为字符串返回，否则返回一个ProcessExecThread对象。</p>

<p><strong>交互问题</strong></p>

<p>&nbsp;&nbsp;&nbsp; Java标准输入输出控制台，运行存在交互的程序可能会存在问题。</p>

<p><strong>传入变量</strong></p>

<p>&nbsp;&nbsp;&nbsp; 进程是在其它线程中异步执行的，当前</p>

<p><strong>事件：</strong></p>

<ul>
	<li><strong>onOut(byte[] bytes, int length, ProcessExecThread pt)</strong><br />
	可以通过重写onOut方法监听进程的输出。<br />
	&nbsp;</li>
	<li><strong>onErrOut(byte[] bytes, int length, ProcessExecThread pt)</strong><br />
	可以通过重写onErrOut方法监听程序的错误输出。<br />
	&nbsp;</li>
	<li><strong>onExited(String out, int exitValue, ProcessExecThread pt)</strong><br />
	可以重写onFinished方法监听进程的退出，其中out是进程输出的字符串（包括err输出，有长度限制）。</li>
</ul>

<p><strong>向进程发信息</strong></p>

<p>&nbsp;&nbsp;&nbsp; 通过ProcessExecThread的方法，异步执行时会返回ProcessExecThread对象。</p>

<ul>
	<li><strong>public void write(byte[] bytes) throws IOException</strong>&nbsp;<br />
	向进写入bytes数组。<br />
	&nbsp;</li>
	<li><strong>public void write(byte[] b, int off, int len) throws IOException</strong><br />
	向进程写入bytes数组，给定偏移量和长度。</li>
</ul>

<p><strong>可以重写的方法：</strong></p>

<p>&nbsp;&nbsp;&nbsp;&nbsp; 上面读取进程输出所列的方法也是可以重写的方法。</p>

<ul>
	<li><strong>boolean isSync()</strong><br />
	是否同步执行。<br />
	&nbsp;</li>
	<li><strong>long getTimeout()</strong><br />
	返回超时时间，单位毫秒。 -1表示永不超时，返回0会设置成默认值10000。<br />
	&nbsp;</li>
	<li><strong>int getMaxContentLenth()</strong><br />
	当进程结束时，进程的输出会当作字符串返回，maxContentLength是返回字符串的最大长度。<br />
	&nbsp;</li>
	<li><strong>File getDir()</strong><br />
	返回工作目录。<br />
	&nbsp;</li>
	<li><strong>String getCmdarray()</strong><br />
	返回要执行的命令，如有多个使用换行符分割。<br />
	&nbsp;</li>
	<li><strong>String getEnvp()</strong><br />
	返回环境变量，如有多个使用换行符分隔。<br />
	&nbsp;</li>
	<li><strong>boolean isProcessManager()</strong><br />
	是否添加到进程管理器中，如果进程管理存在的话。<br />
	&nbsp;</li>
	<li><strong>OutputStream getOutputStream()</strong><br />
	如果返回值不为null，那么进程的输出同时写入该流中。<br />
	&nbsp;</li>
	<li><strong>OutputStream getErrorOutputStream()</strong><br />
	如果返回值不为null，那么进程的错误输出同时会写入该流中。</li>
</ul>
