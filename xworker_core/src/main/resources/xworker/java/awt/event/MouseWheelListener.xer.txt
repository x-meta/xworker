<p><span style="background-color:rgb(255, 255, 255); color:rgb(53, 56, 51)">The listener interface for receiving mouse wheel events on a component. (For clicks and other mouse events, use the </span><code>MouseListener</code><span style="background-color:rgb(255, 255, 255); color:rgb(53, 56, 51)">. <p style="text-align: left;">The class that is interested in processing a mouse wheel event implements this interface (and all the methods it contains).</p>

<p style="text-align: left;">The listener object created from that class is then registered with a component using the component&#39;s <code>addMouseWheelListener</code> method. A mouse wheel event is generated when the mouse wheel is rotated. When a mouse wheel event occurs, that object&#39;s <code>mouseWheelMoved</code>method is invoked.</p>
