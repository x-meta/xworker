<p style="text-align: left;">In order to align a <code>SequentialGroup</code> along the baseline of a baseline aligned <code>ParallelGroup</code> you need to specify which of the elements of the <code>SequentialGroup</code> is used to determine the baseline. The element used to calculate the baseline is specified using one of the <code>add</code>methods that take a <code>boolean</code>. The last element added with a value of <code>true</code> for <code>useAsBaseline</code> is used to calculate the baseline.</p>
